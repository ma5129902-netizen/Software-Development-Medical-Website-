# Medical Website 

## Description
Medical2026 is a healthcare management system developed using ASP.NET Core MVC. The application provides an intuitive interface for managing patients, doctors, appointments, and medical records.

## Features
- User Registration & Login
- Patient Management
- Doctor Management
- Appointment Booking
- Medical Records
- Responsive User Interface

## Technologies
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap

## Team
Developed as a Software Development course project.
